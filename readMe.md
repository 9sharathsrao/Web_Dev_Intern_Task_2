## Task_2
A dynamic To-Do list where users can add, remove, and mark tasks complete.

## Summary of Work
1. Used An '<input>' field for data entry, a '<button>' to trigger the action, and an empty '<ul>' that acts as a placeholder for tasks.
2. Used Flexbox to align the checkbox, task text, and delete button in a single row. Defined the '.completed' class with 'text-decoration: line-through', which provides immediate visual confirmation when a task is finished.
3. Used listens for clicks to grab text from the input and push it to the list.
4. Used a change listener on the checkbox to toggle the "completed" styling.
5. Used a click listener on the "Delete" button to remove the specific '<li>' from the DOM entirely.
6. Ensures the UI updates instantly without a page reload, providing a single page application feel.
